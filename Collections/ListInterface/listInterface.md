# LIST INTERFACE 
- part of javautil package and sub interface of collection interface . 
- it provides a way to store a ordered collection of elements sequentially . 
- list allow for precise control over where elements are inserted and can contain duplicate elements.


# When to use List ? 
- when we want to follow ordered sequence
- or we want to allow duplication

- list interface is implemented by several classes in java collection framework such as ArrayList, LinkedList,Vector and Stack
